# Python Daily Practice from Scratch

This repository is dedicated to consistent daily practice in Python, focusing on improving problem-solving skills and strengthening core programming fundamentals. It contains a comprehensive collection of notebooks and Python scripts covering Python basics, data structures, and Object-Oriented Programming (OOP) concepts.

## Repository Overview

- **Language Composition:** 94.7% Jupyter Notebook, 5.3% Python
- **Purpose:** Daily learning and practice through hands-on coding examples

## 📁 Repository Structure

### Directories

- **`DSA_Practice_from_basic_to_advanced/`** - Data Structures and Algorithms practice materials organized from basic to advanced levels

### Core Python Concepts

#### Data Structures
- **`python_lists.ipynb`** - Comprehensive guide to Python lists, operations, and methods
- **`python_list_problems.ipynb`** - Practice problems focused on list manipulation
- **`python_tuple.ipynb`** - Understanding tuples, immutability, and use cases
- **`python_dict.ipynb`** - Dictionary operations, key-value pairs, and dictionary methods
- **`python_set.ipynb`** - Set operations and set-related problems
- **`python_codes.ipynb`** - General Python code examples and snippets

#### Object-Oriented Programming (OOP)
- **`python_oops_notes.ipynb`** - Comprehensive OOP notes covering all major concepts
- **`Python_oops_basic.py`** - Basic OOP principles and fundamentals
- **`python_constructor.py`** - Constructor patterns and object initialization
- **`python_inheritance.py`** - Inheritance concepts and implementation
- **`python_inheritance_types.ipynb`** - Different types of inheritance in detail
- **`python_Polymorphism.ipynb`** - Polymorphism concepts and examples
- **`python_encapsulation.py`** - Encapsulation and data hiding
- **`python_abstraction.py`** - Abstract classes and abstraction concepts

#### Practice Projects
- **`python_small_constructor_project.py`** - Small practical project using constructors
- **`python_practice_questions.ipynb`** - Collection of practice questions and solutions

## 🎯 Learning Path

This repository progresses through:
1. **Fundamentals** - Basic Python syntax and operations
2. **Data Structures** - Lists, tuples, dictionaries, sets
3. **Problem Solving** - Practice problems on data structures
4. **OOP Concepts** - Classes, inheritance, polymorphism, encapsulation, abstraction
5. **Advanced DSA** - Data structures and algorithms from basic to advanced

## 📝 Notes

- Most content is organized in **Jupyter Notebooks** for interactive learning
- Some foundational concepts are provided as **Python scripts** for quick reference
- Each file focuses on specific concepts with practical examples

## 🚀 How to Use

1. Clone the repository
2. Open Jupyter notebooks with `jupyter notebook` or use an online Jupyter environment
3. Follow the files in the order listed or navigate to topics of interest
4. Practice with the provided examples and complete the practice problems
5. Explore the DSA directory for algorithm-focused problems
